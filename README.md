# EAstock first approach to an Android app
Expert Advisor Stock quotes, a trading system to analyse data from yahoo finances by data mining.

## Esto es
Informar del estado de los índices que interesan al usuario de la bolsa y ofrecer la posibilidad de visualizar 
un valor de forma detallada. Guiar al usuario a la hora de invertir, operando en tiempo "real".

## Características
*	Gestión de actividades mediante pestañas.
*	Gestión de tickers mediante proveedor de búsqueda.
*	Refresco de datos manual y automático cada 15 segundos (tiempo real).
*	Servicio de notificaciones en la barra de notificaciones.
*	Permisos para almacenamiento, comunicación en red y control de hardware.
*	Elección del eje temporal de la vista en detalle.

## Mejoras futuras
*	Optimización para conseguir mayor velocidad.
*	Gráficas dinámicas en tiempo real.
*	Múltiples carteras.
*	Mejoras de los filtros de análisis bursátil, así como la adicción de nuevos.
*	Elección de mercados a comparar.
*	Inclusión de más tickers a nivel mundial.

	Feel free to try implementing those and send me a pull request!
